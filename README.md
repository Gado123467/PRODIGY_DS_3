# Bank Dataset 

## Overview

The Bank Marketing dataset consists of 45,211 rows and 17 columns, providing a rich and structured source of customer demographic, financial, and campaign-related information suitable for classification and predictive analysis.
## Objectives

* Explore the bank dataset to understand customer demographics and behaviors
* Visualize categorical variables (e.g., job, marital status, housing loan) using bar charts
* Visualize continuous variables (e.g., age, account balance) using histograms
* Gain insights into patterns that may influence banking decisions

## Dataset

The dataset includes the following key columns:

* `age`: Age of the client
* `job`: Job type (admin, technician, etc.)
* `marital`: Marital status (single, married, divorced)
* `education`: Education level
* `default`: Has credit in default (yes/no)
* `balance`: Average yearly balance in euros
* `housing`: Has housing loan (yes/no)
* `loan`: Has personal loan (yes/no)
* `contact`: Contact communication type (cellular, telephone)
* `day`, `month`, `duration`: Last contact date and duration
* `campaign`, `pdays`, `previous`, `poutcome`: Marketing campaign data
* `y`: Outcome of the marketing campaign (yes/no)
